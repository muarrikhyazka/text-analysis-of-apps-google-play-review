# Text Analysis of Apps Google Play Reviews

This repository contains a Python script that can be used to analyze the text reviews of mobile apps from the Google Play Store. By using this script, you can gain valuable insights into the opinions and sentiments of users who have downloaded and used your app. The script is easy to use, and it generates useful visualizations such as pie charts, bar charts, and word clouds to help you understand the reviews better. This tool can be especially useful for app developers and marketers who want to improve their app's user experience and reputation on the Google Play Store.
